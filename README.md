# Spotify Popularity Prediction

## Overview
This project predicts Spotify track popularity using audio features such as energy, danceability, and loudness.

## Problem
Can audio features alone predict a song’s popularity?

## Models Used
- Linear Regression
- Random Forest Regressor

## Metrics
- RMSE
- R² Score

## Dataset
https://huggingface.co/datasets/maharshipandya/spotify-tracks-dataset
